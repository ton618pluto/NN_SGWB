Superimposed V5 Pipeline

简要说明从超参数采样到带噪声训练数据生成，再到 timeflow_v5 训练的文件级数据流。

整体流程图

1. 超参数采样

先定义一批总体分布参数，作为后续所有样本组的起点。

2. 事件参数生成

在每组超参数下，采样出一批 CBC 事件的物理参数。

3. 纯信号构造

把事件参数投影成探测器中的纯引力波时域信号。

↓

4A. 纯信号支路

保留按超参数组和样本编号组织好的纯信号样本。

与

4B. 纯噪声支路

独立生成探测器噪声，并按编号准备好可复用的噪声段。

↓

5. 信号与噪声叠加

按样本编号把纯信号和对应噪声一一相加，形成带噪观测。

6. 切分与整理

把长时序切成训练片段，并把标签一起保存成训练可读格式。

7. 模型训练

读取整理后的双探测器数据，切分训练集并训练 TimeFlow V5。

draw_hyperparameters_v2.py

作用
采样联合超参数，生成后续事件采样要用的超参数表。
输入
无外部输入，脚本内部定义采样范围和随机种子。
输出
  • scripts/joint_hyperparams_train/v3/joint_hyperparams_train_v3.npz

draw_CBC_params_diff.py

作用
读取超参数表，为每个超参数组生成一份 CBC 事件参数文件。
输入
  • scripts/joint_hyperparams_train/v3/joint_hyperparams_train_v3.npz
输出
  • scripts/parameter_sampling_train/v3/CBC_params_examplexxxxx.npz

generate_pure_cbc_frames_sampling5.py

作用
把每组 CBC 事件参数变成纯信号 .gwf 文件，并生成标签表。
输入
  • scripts/parameter_sampling_train/v3/CBC_params_examplexxxxx.npz
输出
  • scripts/training_set/v3/training_set0/H1/*.gwf
  • scripts/training_set/v3/training_set0/L1/*.gwf
  • scripts/training_set/v3/training_set0/train_idx.csv

generate_pure_noise_frames.py

作用
生成 H1 和 L1 的纯噪声 .gwf 文件。
输入
  • scripts/O4_mock.txt
输出
  • scripts/noise_waveform_H1/*.gwf
  • scripts/noise_waveform_L1/*.gwf

superimposed_friendly.py

作用
把纯信号和纯噪声按编号叠加，得到带噪声的 .gwf。
输入
  • scripts/training_set/v3/training_set0/H1/*.gwf
  • scripts/training_set/v3/training_set0/L1/*.gwf
  • scripts/noise_waveform_H1/*.gwf
  • scripts/noise_waveform_L1/*.gwf
输出
  • scripts/training_set_superimposed/v0/H1/*.gwf
  • scripts/training_set_superimposed/v0/L1/*.gwf

processData_superimposed.py

作用
读取叠加后的 .gwf，切成训练用 .npy 小段，并保留标签。
输入
  • scripts/training_set_superimposed/v0/H1/*.gwf
  • scripts/training_set_superimposed/v0/L1/*.gwf
  • scripts/training_set_superimposed/v0/train_idx.csv
输出
  • scripts/processed_data_superimposed/v0/H1_splits/*.npy
  • scripts/processed_data_superimposed/v0/L1_splits/*.npy

timeflow_v5/train_flow.py

作用
读取切分后的带噪声训练数据，进行 train / val / test 切分并训练 V5 模型。
输入
  • scripts/processed_data_superimposed/v0/H1_splits/*.npy
  • scripts/processed_data_superimposed/v0/L1_splits/*.npy
输出
  • scripts/timeflow_v5/outputs/best_flow_v5.pt
  • scripts/timeflow_v5/outputs/run_config.json
  • scripts/timeflow_v5/outputs/data_split.json
