# NN_SGWB
Detecting SGWB using NN.

## scripts:
**draw_CBC_params.ipynb**: an example notebook to draw CBC parameters for injection

**generate_pure_cbc_frames.py**: python script to inject CBC events into time series

**generate_pure_noise_frames.py**: python script to inject noise into time series

**generate_pure_SGWB_frames.py**: python script to inject SGWB into time series

**combine_frames.py**: python script to combine frames containing different injections

**plot_gwf.ipynb**: a notebook to plot simulated gravitational-wave frames (gwf)
