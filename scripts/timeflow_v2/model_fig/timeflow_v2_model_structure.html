Time-Domain SGWB Inference

TimeFlow V2 模型结构展示

该模型保留原始时域输入，使用双探测器 H1 + L1 波形作为条件信息， 先通过残差一维卷积编码器提取长时序特征，再用条件归一化流学习 p(θ | x)，输出 10 维种群超参数的联合后验分布。

Input Length 524288
Input Channels H1 + L1
Context Dim 256
Posterior Dim 10

整体流程

从切分好的 `.npy` 原始时域样本出发，经过数据预处理、时域编码、条件 flow 建模， 最终得到 10 维物理超参数的联合后验。

Input

原始波形输入

读取 H1 / L1 切片后的 `.npy` 文件，单个 segment 长度为 524288。

Preprocess

数据预处理

每个通道乘以 `1e23` 后再做去均值；标签删除原始第 3 个分量并做 z-score 标准化。

Encoder

TemporalEncoder1D

通过 stem + 7 个残差卷积块，逐步压缩长度并扩大感受野，提取全局时域证据。

Flow

Conditional Flow

用 6 层随机置换 + 自回归样条变换，对 10 维参数联合分布进行条件建模。

Output

后验输出

训练时最小化 NLL；推理时从 `p(θ | x)` 中采样得到参数后验样本。

编码器结构

TemporalEncoder1D

该编码器替代了旧版的 `flatten + 大全连接` 结构。核心思路是： 用更深的时域残差网络建立跨长时间尺度的表征，再用全局池化汇聚统计信息。

Stem
Conv1d(`in_channels → 24`, kernel=15, stride=2) + GroupNorm + GELU
Block 1
ResidualConvBlock(`24 → 32`, stride=2, dilation=1)
Block 2
ResidualConvBlock(`32 → 48`, stride=2, dilation=1)
Block 3
ResidualConvBlock(`48 → 64`, stride=2, dilation=2)
Block 4
ResidualConvBlock(`64 → 96`, stride=2, dilation=2)
Block 5
ResidualConvBlock(`96 → 128`, stride=2, dilation=4)
Block 6
ResidualConvBlock(`128 → 160`, stride=2, dilation=4)
Block 7
ResidualConvBlock(`160 → 192`, stride=2, dilation=8)

池化与投影

经过残差块之后，模型不再使用超大的展平向量，而是直接做全局池化来得到紧凑、稳定的 高层语义表示。

Avg Pool
AdaptiveAvgPool1d(1) 提取全局平均统计量
Max Pool
AdaptiveMaxPool1d(1) 保留显著峰值响应
Concat
将两路池化结果拼接成 384 维特征
Projector
Linear(384→256) + GELU + LayerNorm + Linear(256→context_dim)
默认 `context_dim = 256`
支持单通道 / 双通道输入
更适合长时域原始信号

条件归一化流

GWFlowModelV2

Flow 部分以编码器输出的 context 为条件，对 10 维参数向量 `θ` 建立联合后验模型。 这样不仅能给出点估计，还能直接给出参数不确定度与参数间相关性。

Base Distribution = StandardNormal(10)
Flow Layers = 6
Flow Hidden = 256
Spline Bins = 8
Tail Bound = 3.0
for each flow layer: RandomPermutation(features=10) MaskedPiecewiseRationalQuadraticAutoregressiveTransform( features=10, hidden_features=256, context_features=256, num_blocks=2, use_residual_blocks=True, tails="linear", tail_bound=3.0, num_bins=8 )

训练与推理方式

训练阶段，模型接收 `(theta, x)`，直接返回负对数似然作为 loss； 推理阶段，给定一个波形样本 `x`，从 `p(θ | x)` 中采样，得到完整后验样本集。

Train
`loss = -log p(theta | x)`，最小化 NLL
Infer
`theta_samples ~ p(theta | x)`，用于画 posterior
Optimizer
AdamW + ReduceLROnPlateau + gradient clipping
AMP
CUDA 下使用 autocast 与 GradScaler
